# Poker Score App

一个用于本地记录和管理牌局得分的轻量级 Kivy 应用。主要功能：

- 录入页面：为每位玩家输入手牌/惩罚（dun），支持通过用户名长按拖拽整行重排（改变本局名次顺序）。
- 保存每局时同时保存两种名次：拖拽顺序（ranks）与按分数计算的名次（ranks_by_score）。
- 得分页面：显示每局得分、名次与奖杯标识（按保存的名次显示）。
- 已添加一个空的“统计”页面，便于后续扩展统计功能。

快速开始

1. 安装依赖（推荐使用虚拟环境）

```bash
python -m venv .venv
source .venv/bin/activate
pip install kivy
# 若需要更多依赖，请按需要安装
```

2. 运行应用：

```bash
python main.py
```

文件与位置

- `main.py` - 应用主文件（包含界面、逻辑、数据读写）。
- `score_data.json` - 默认的数据文件（会在保存时写入）。

已知注意事项与后续改进

- 建议在桌面环境或移动设备上运行 Kivy 应用以获得最佳体验。
- 目前拖拽与输入框的交互已实现并保留输入值，但仍建议手动测试边界情况（多点触控、快速点击等）。
- 建议为 CI 增加更多检查（例如格式化、单元测试）并补充统计页面内容。
